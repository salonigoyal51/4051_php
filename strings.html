String Handling in PHP

Strings are one of the most common data types in PHP and are used to store sequences of characters. PHP provides powerful built-in functions to manipulate, compare, and modify strings easily.

String handling functions allow you to perform operations such as concatenation, searching, replacing, and measuring length. These operations are essential for tasks like form validation, text processing, and data formatting.

PHP supports both single-quoted and double-quoted strings, each behaving slightly differently. Double-quoted strings interpret variables and special characters, while single-quoted strings treat them literally.

String handling in PHP refers to performing different operations on strings such as concatenation, comparison, modification, and searching. A string is a sequence of characters enclosed in single (' ') or double (" ") quotes.

1. Declaring Strings:
Strings can be created using single or double quotes.
Example:
<?php
$str1 = 'Hello';
$str2 = "World";
echo $str1 . " " . $str2;
?>

Output:
Hello World

2. String Concatenation:
In PHP, the dot (.) operator is used to join two or more strings together.
Example:
<?php
$name = "Saloni";
echo "Hello " . $name . "!";
?>

Output:
Hello Saloni!

3. String Functions in PHP:
PHP provides many built-in string functions to perform various tasks like measuring length, changing case, finding words, and replacing text.

a) strlen() – Returns the length of a string.
Example:
<?php echo strlen("Hello PHP"); ?>
Output: 9

b) strrev() – Reverses a string.
Example:
<?php echo strrev("Hello"); ?>
Output: olleH

c) strtolower() – Converts all characters to lowercase.
Example:
<?php echo strtolower("WELCOME"); ?>
Output: welcome

d) strtoupper() – Converts all characters to uppercase.
Example:
<?php echo strtoupper("hello"); ?>
Output: HELLO

e) ucfirst() – Converts the first character of a string to uppercase.
Example:
<?php echo ucfirst("php language"); ?>
Output: Php language

f) ucwords() – Converts the first character of each word to uppercase.
Example:
<?php echo ucwords("welcome to php"); ?>
Output: Welcome To Php

g) trim() – Removes whitespace or predefined characters from both sides of a string.
Example:
<?php
$text = " Hello PHP ";
echo trim($text);
?>
Output: Hello PHP

h) str_replace() – Replaces some characters with others in a string.
Example:
<?php echo str_replace("PHP", "World", "Hello PHP"); ?>
Output: Hello World

i) strpos() – Finds the position of the first occurrence of a substring in a string.
Example:
<?php echo strpos("Hello World", "World"); ?>
Output: 6

j) substr() – Returns a part of a string.
Example:
<?php echo substr("Welcome to PHP", 0, 7); ?>
Output: Welcome

4. Heredoc and Nowdoc Syntax:
These are used to create multi-line strings easily.

Heredoc Example:
<?php
$str = <<<TEXT
This is a multi-line
string using Heredoc syntax.
TEXT;
echo $str;
?>

Nowdoc Example:
<?php
$str = <<<'TEXT'
This is a Nowdoc string.
TEXT;
echo $str;
?>

5. Comparing Strings:
You can compare two strings using comparison operators (==, !=, ===) or the strcmp() function.
Example:
<?php
echo strcmp("Hello", "Hello"); // Output: 0 (equal)
echo strcmp("Apple", "Banana"); // Output: negative value
?>

6. String Interpolation:
When using double quotes, variables inside a string are automatically replaced with their values.
Example:
<?php
$name = "Saloni";
echo "Hello $name";
?>
Output: Hello Saloni

7. Example Program:
<?php
$text = "PHP is Fun!";
echo "Original: $text<br>";
echo "Length: " . strlen($text) . "<br>";
echo "Uppercase: " . strtoupper($text) . "<br>";
echo "Reversed: " . strrev($text);
?>

Output:
Original: PHP is Fun!
Length: 11
Uppercase: PHP IS FUN!
Reversed: !nuF si PHP

8. Summary:
String handling in PHP is essential for text processing. PHP provides several built-in string functions such as strlen(), strrev(), strtoupper(), strtolower(), str_replace(), and substr() that make working with text easy and efficient. These functions are widely used in web development for form processing, data formatting, and dynamic content generation.

Example Programs

Program 1: String Concatenation

<?php
$first = "Learn";
$second = "PHP";
$result = $first . " " . $second;
echo $result;
?>

Output:

Learn PHP

Program 2: String Length

<?php
$str = "Hello World";
echo "The length of the string is: " . strlen($str);
?>

Output:

The length of the string is: 11

Program 3: Counting Words in a String

<?php
$text = "PHP makes web development easier";
echo "Word count: " . str_word_count($text);
?>

Output:

Word count: 5

Program 4: Reversing a String

<?php
$str = "Saloni";
echo "Reversed string: " . strrev($str);
?>

Output:

Reversed string: inolaS

Program 5: Replacing Text within a String

<?php
$sentence = "I love Java.";
$newSentence = str_replace("Java", "PHP", $sentence);
echo $newSentence;
?>

Output:

I love PHP.

Summary

String handling in PHP is essential for managing textual data efficiently. With built-in functions like strlen(), strrev(), str_replace(), and str_word_count(), you can easily manipulate strings without writing complex code.

Mastering string functions allows developers to create dynamic, user-friendly applications involving text processing, data validation, and formatting.
