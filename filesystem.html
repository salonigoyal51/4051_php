Working with the File System in PHP

PHP allows developers to work directly with files on the server — reading, writing, creating, deleting, and uploading files. This feature is essential for dynamic content management, data storage, and user uploads.

The main functions used for file handling are fopen(), fwrite(), fread(), fclose(), file_exists(), and unlink().

File handling makes PHP a powerful tool for managing documents, logs, and multimedia files in web applications.

PHP provides powerful functions to interact with the file system. It allows developers to create, open, read, write, append, and delete files on the server. Working with files is useful for tasks like storing data, managing logs, or processing uploaded files.

1. Opening a File:
The fopen() function is used to open a file. It requires two parameters: the file name and the mode of operation.
Syntax: fopen(filename, mode);
Example:
$file = fopen("data.txt", "r");

Common File Modes:
"r" – Open file for reading (file must exist).
"w" – Open file for writing (creates a new file or overwrites an existing one).
"a" – Open file for appending (writes data at the end of the file).
"x" – Create a new file and open for writing (gives error if file already exists).
"r+" – Open for reading and writing (file must exist).
"w+" – Open for reading and writing (creates a new file or overwrites existing one).

2. Reading from a File:
To read data from a file, PHP provides functions like fread(), fgets(), and fgetc().
Example:
$file = fopen("data.txt", "r");
echo fread($file, filesize("data.txt"));
fclose($file);

3. Writing to a File:
The fwrite() function is used to write data to a file.
Example:
$file = fopen("data.txt", "w");
fwrite($file, "Welcome to PHP File Handling!");
fclose($file);

4. Appending Data to a File:
To add new content without deleting existing data, open the file in append mode ("a").
Example:
$file = fopen("data.txt", "a");
fwrite($file, "\nThis line is added later.");
fclose($file);

5. Checking if a File Exists:
Use the file_exists() function to check whether a file is available before opening it.
Example:
if (file_exists("data.txt")) { echo "File exists."; } else { echo "File not found."; }

6. Deleting a File:
The unlink() function is used to delete an existing file from the server.
Example:
if (unlink("data.txt")) { echo "File deleted successfully."; } else { echo "Error deleting file."; }

7. Reading File Line by Line:
To read large files efficiently, use fgets() in a loop.
Example:
$file = fopen("data.txt", "r");
while (!feof($file)) { echo fgets($file) . "<br>"; }
fclose($file);

8. Reading Entire File with file_get_contents():
You can read the whole file into a string using one line of code.
Example:
$content = file_get_contents("data.txt");
echo $content;

9. Writing Entire File with file_put_contents():
This function writes a string directly to a file (it creates the file if it doesn’t exist).
Example:
file_put_contents("data.txt", "PHP makes file handling easy!");

10. Example Program:
<?php
$file = fopen("example.txt", "w");
fwrite($file, "Learning PHP File System Functions.");
fclose($file);

$file = fopen("example.txt", "r");
echo fread($file, filesize("example.txt"));
fclose($file);
?>

Output:
Learning PHP File System Functions.

11. Important File Handling Functions in PHP:
fopen() – Opens a file.
fclose() – Closes an open file.
fread() – Reads from a file.
fwrite() – Writes to a file.
fgets() – Reads a single line from a file.
feof() – Checks if the end of the file has been reached.
unlink() – Deletes a file.
file_exists() – Checks if a file exists.
file_get_contents() – Reads the entire file into a string.
file_put_contents() – Writes data to a file.

12. Summary:
PHP’s file handling functions make it possible to create, read, write, append, and delete files easily. The functions like fopen(), fread(), fwrite(), and fclose() are commonly used for basic operations. Using these, developers can store logs, manage content, and handle user data efficiently through the file system.

Example Programs

Program 1: Creating and Writing to a File

<?php
$file = fopen("example.txt", "w") or die("Unable to open file!");
$text = "Hello, this is a file created by PHP.";
fwrite($file, $text);
fclose($file);
echo "File created and written successfully!";
?>

Output:

File created and written successfully!

Program 2: Reading Data from a File

<?php
$file = fopen("example.txt", "r") or die("Unable to open file!");
echo fread($file, filesize("example.txt"));
fclose($file);
?>

Output:

Hello, this is a file created by PHP.

Program 3: Appending Data to a File

<?php
$file = fopen("example.txt", "a") or die("Unable to open file!");
$text = "\nThis line is appended using PHP.";
fwrite($file, $text);
fclose($file);
echo "Data appended successfully!";
?>

Output:

Data appended successfully!

Program 4: Checking if a File Exists

<?php
$filename = "example.txt";
if (file_exists($filename)) {
    echo "File '$filename' exists!";
} else {
    echo "File does not exist!";
}
?>

Output:

File 'example.txt' exists!

Program 5: Deleting a File

<?php
$filename = "example.txt";
if (unlink($filename)) {
    echo "File deleted successfully!";
} else {
    echo "Error deleting file.";
}
?>

Output:

File deleted successfully!

Summary

PHP provides a variety of file-handling functions to manage files on the server. You can create, read, write, append, and delete files dynamically using built-in PHP methods.

Understanding file operations helps developers build applications that handle user uploads, manage logs, and create dynamic reports efficiently.
