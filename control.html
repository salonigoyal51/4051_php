Control Structures in PHP

Control structures are one of the most essential parts of any programming language. In PHP, they allow developers to control the flow of code execution based on conditions and repetitions. These structures make decisions (like whether to run a certain block of code) and repeat tasks until specific conditions are met. PHP supports a variety of control statements such as if, if...else, if...elseif...else, switch, and different types of loops like for, while, and foreach.

Control structures are vital for creating dynamic logic-driven web applications. They help automate repetitive processes, make logical decisions, validate user inputs, and manage real-time workflows. Without control structures, PHP would execute instructions linearly, making dynamic behavior impossible.

Control structures in PHP are statements that determine the flow of execution in a program. They help in making decisions, repeating actions, and controlling the logic based on conditions. The main types of control structures in PHP are Conditional Statements and Looping Statements.

1. Conditional Statements: Conditional statements are used to perform different actions based on different conditions. PHP supports several types of conditional statements.

a) if Statement: It is used to execute a block of code only if the given condition is true.
Example:
if ($age >= 18) { echo "You are eligible to vote."; }

b) if...else Statement: It executes one block of code if the condition is true, otherwise executes another block.
Example:
if ($marks >= 40) { echo "Pass"; } else { echo "Fail"; }

c) if...elseif...else Statement: It is used when you need to check multiple conditions.
Example:
if ($marks >= 90) { echo "Grade A"; } elseif ($marks >= 75) { echo "Grade B"; } else { echo "Grade C"; }

d) switch Statement: It is used to select one block of code to execute from multiple options. It works like multiple if-else statements but in a cleaner way.
Example:
switch ($day) {
case "Monday": echo "Start of the week"; break;
case "Friday": echo "Weekend is near"; break;
default: echo "Midweek day";
}

2. Looping Statements: Loops are used to execute a block of code repeatedly as long as a certain condition is true. PHP provides several types of loops.

a) while Loop: Executes the block of code as long as the condition is true.
Example:
$x = 1;
while ($x <= 5) { echo "Number: $x <br>"; $x++; }

b) do...while Loop: Similar to while loop but the condition is checked after executing the block once. The loop will always run at least once.
Example:
$x = 1;
do { echo "Number: $x <br>"; $x++; } while ($x <= 5);

c) for Loop: Used when you know in advance how many times to execute the code.
Example:
for ($i = 1; $i <= 5; $i++) { echo "Count: $i <br>"; }

d) foreach Loop: Special loop used for arrays. It iterates through each element in an array.
Example:
$colors = array("Red", "Green", "Blue");
foreach ($colors as $color) { echo $color . "<br>"; }

3. Jump Statements: PHP provides jump statements like break, continue, and exit to control the flow inside loops or scripts.

a) break: Terminates the current loop or switch statement.
b) continue: Skips the rest of the current iteration and moves to the next iteration.
c) exit or die: Terminates the script execution completely.

Example of break and continue:
for ($i = 1; $i <= 5; $i++) {
if ($i == 3) continue;
if ($i == 5) break;
echo "Number: $i <br>";
}

Summary: Control structures determine how a program executes. Conditional statements like if and switch decide which code runs based on conditions. Looping structures like while, do...while, for, and foreach are used for repetition. Jump statements such as break, continue, and exit help control the execution flow inside loops and programs.

Example Programs

Program 1: Simple if Statement

<?php
$age = 18;
if ($age >= 18) {
    echo "You are eligible to vote.";
}
?>

Output:

You are eligible to vote.

Program 2: if...else Statement

<?php
$marks = 40;
if ($marks >= 50) {
    echo "You passed the exam!";
} else {
    echo "You failed the exam!";
}
?>

Output:

You failed the exam!

Program 3: if...elseif...else Ladder

<?php
$score = 82;
if ($score >= 90) {
    echo "Grade: A+";
} elseif ($score >= 75) {
    echo "Grade: A";
} elseif ($score >= 60) {
    echo "Grade: B";
} else {
    echo "Grade: C";
}
?>

Output:

Grade: A

Program 4: switch Statement

<?php
$day = "Tuesday";
switch ($day) {
    case "Monday":
        echo "Start of the work week!";
        break;
    case "Tuesday":
        echo "Second day of the week!";
        break;
    case "Friday":
        echo "Weekend is coming!";
        break;
    default:
        echo "Just another day.";
}
?>

Output:

Second day of the week!

Program 5: Using Loops (for, while, foreach)

<?php
// For loop example
echo "For Loop: ";
for ($i = 1; $i <= 5; $i++) {
    echo $i . " ";
}

// While loop example
echo "<br>While Loop: ";
$j = 1;
while ($j <= 3) {
    echo $j . " ";
    $j++;
}

// Foreach loop example
echo "<br>Foreach Loop: ";
$fruits = array("Apple", "Banana", "Cherry");
foreach ($fruits as $fruit) {
    echo $fruit . " ";
}
?>

Output:

For Loop: 1 2 3 4 5 
While Loop: 1 2 3 
Foreach Loop: Apple Banana Cherry

Summary

In this section, we learned how PHP’s control structures determine the flow of a program. Conditional statements like if, else, and switch help in making decisions based on specific conditions, while looping structures like for, while, and foreach allow repetition of tasks. These are fundamental in building logical web applications — such as validating form data, managing sessions, or dynamically displaying content.

By mastering control structures, developers can make their PHP applications more intelligent, interactive, and efficient.
