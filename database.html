Integrating PHP with Database

PHP can connect and interact with databases such as MySQL to store, retrieve, update, and delete data dynamically. This allows developers to create interactive web applications like login systems, product catalogs, and dashboards.

PHP uses extensions like MySQLi (MySQL Improved) and PDO (PHP Data Objects) to connect with databases. These provide secure and efficient methods for executing SQL queries.

Database integration forms the backbone of dynamic websites, making PHP one of the most popular languages for backend development.

Integrating PHP with a database allows web applications to store, retrieve, and manage data dynamically. The most commonly used database with PHP is MySQL because it is fast, reliable, and easy to use.

1. What is PHP and Database Integration?
PHP and MySQL integration means connecting a PHP script to a MySQL database so that PHP can send queries to the database and receive results. It allows the creation of dynamic websites like login systems, registration forms, and data management systems.

2. Steps to Connect PHP with MySQL Database:

Step 1: Create a Database
Open phpMyAdmin or MySQL console and create a database using the following command:
CREATE DATABASE studentdb;

Step 2: Create a Table
Inside the database, create a table to store data:
CREATE TABLE students (id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(50), age INT, course VARCHAR(50));

Step 3: Establish Connection in PHP
Use the mysqli_connect() function to connect PHP with MySQL.
Example:
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "studentdb";

$conn = mysqli_connect($servername, $username, $password, $dbname);
if (!$conn) { die("Connection failed: " . mysqli_connect_error()); }
echo "Connected successfully";

Step 4: Insert Data into Table
After connection, use an SQL INSERT statement to add records.
Example:
$sql = "INSERT INTO students (name, age, course) VALUES ('Riya', 20, 'BSc')";
if (mysqli_query($conn, $sql)) { echo "Record inserted successfully"; } else { echo "Error: " . mysqli_error($conn); }

Step 5: Retrieve Data from Table
Use SELECT query to fetch data.
Example:
$result = mysqli_query($conn, "SELECT * FROM students");
while ($row = mysqli_fetch_assoc($result)) {
echo "ID: " . $row['id'] . " Name: " . $row['name'] . " Age: " . $row['age'] . " Course: " . $row['course'] . "<br>";
}

Step 6: Close the Connection
After performing operations, close the database connection using:
mysqli_close($conn);

3. Commonly Used MySQLi Functions:
mysqli_connect() – Opens a new connection to the MySQL server.
mysqli_query() – Performs a query on the database.
mysqli_fetch_assoc() – Fetches a result row as an associative array.
mysqli_num_rows() – Returns the number of rows in a result set.
mysqli_close() – Closes the database connection.

4. Example Program:
<?php
$conn = mysqli_connect("localhost", "root", "", "studentdb");
if (!$conn) { die("Connection failed: " . mysqli_connect_error()); }
$sql = "SELECT * FROM students";
$result = mysqli_query($conn, $sql);
while ($row = mysqli_fetch_assoc($result)) {
echo $row['id'] . " - " . $row['name'] . " - " . $row['course'] . "<br>";
}
mysqli_close($conn);
?>

5. Advantages of PHP-MySQL Integration:
- Enables dynamic and interactive web pages.
- Data can be stored, retrieved, and updated easily.
- Supports user management systems like login and registration.
- Provides secure and structured data storage.

6. Summary:
Integrating PHP with a database allows PHP scripts to communicate with MySQL. Using functions like mysqli_connect(), mysqli_query(), and mysqli_fetch_assoc(), developers can store, retrieve, and manage data efficiently. This integration forms the foundation for most dynamic web applications such as blogs, e-commerce websites, and student management systems.

Example Programs

Program 1: Connect PHP with MySQL Database

<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "school";

// Create connection
$conn = mysqli_connect($servername, $username, $password, $dbname);

// Check connection
if (!$conn) {
    die("Connection failed: " . mysqli_connect_error());
}
echo "Connected successfully!";
?>

Output:

Connected successfully!

Program 2: Creating a Table in MySQL

<?php
$conn = mysqli_connect("localhost", "root", "", "school");

$sql = "CREATE TABLE Students (
id INT(6) AUTO_INCREMENT PRIMARY KEY,
name VARCHAR(30) NOT NULL,
age INT(3),
grade VARCHAR(5)
)";

if (mysqli_query($conn, $sql)) {
    echo "Table Students created successfully";
} else {
    echo "Error creating table: " . mysqli_error($conn);
}
?>

Output:

Table Students created successfully

Program 3: Inserting Data into Table

<?php
$conn = mysqli_connect("localhost", "root", "", "school");

$sql = "INSERT INTO Students (name, age, grade)
VALUES ('Saloni', 19, 'A')";

if (mysqli_query($conn, $sql)) {
    echo "Record added successfully!";
} else {
    echo "Error: " . mysqli_error($conn);
}
?>

Output:

Record added successfully!

Program 4: Retrieving Data from Table

<?php
$conn = mysqli_connect("localhost", "root", "", "school");

$result = mysqli_query($conn, "SELECT * FROM Students");

while ($row = mysqli_fetch_assoc($result)) {
    echo "Name: " . $row["name"] . " | Age: " . $row["age"] . " | Grade: " . $row["grade"] . "<br>";
}
?>

Output:

Name: Saloni | Age: 19 | Grade: A
Name: Riya | Age: 18 | Grade: B

Program 5: Updating Data in Table

<?php
$conn = mysqli_connect("localhost", "root", "", "school");

$sql = "UPDATE Students SET grade='A+' WHERE name='Saloni'";

if (mysqli_query($conn, $sql)) {
    echo "Record updated successfully!";
} else {
    echo "Error updating record: " . mysqli_error($conn);
}
?>

Output:

Record updated successfully!

Summary

PHP’s integration with MySQL enables web developers to store and manage data dynamically. From connecting to creating tables and retrieving or updating records, PHP makes database operations easy and efficient.

Understanding how to interact with a database is essential for building data-driven applications such as registration systems, e-commerce sites, and content management platforms.
