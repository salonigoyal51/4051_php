Functions in PHP

Functions in PHP are reusable blocks of code that perform specific tasks. Instead of writing the same logic multiple times, you can define a function once and call it whenever needed. Functions improve readability, reduce redundancy, and make code easier to maintain.

PHP supports both built-in functions (like strlen(), date(), and array_push()) and user-defined functions that you can create according to your project’s needs. A function can accept parameters, perform operations, and return values. Using functions also makes large projects modular, where each part of the code can be tested or modified independently.

Functions in PHP are reusable blocks of code that perform a specific task. They help make programs modular, easy to read, and reduce repetition. Instead of writing the same code multiple times, you can create a function and call it whenever needed.

1. Defining a Function:
A function in PHP is defined using the function keyword, followed by the function name, parentheses, and a block of code.
Syntax:
function functionName() {
  // code to be executed
}

Example:
<?php
function greet() {
  echo "Hello, welcome to PHP Functions!";
}
greet(); // Function call
?>

Output:
Hello, welcome to PHP Functions!

2. Function with Parameters:
Parameters are variables that are passed into a function. They allow you to provide input to a function.
Example:
<?php
function add($a, $b) {
  echo "Sum: " . ($a + $b);
}
add(5, 10);
?>

Output:
Sum: 15

3. Function with Return Value:
A function can return a value using the return statement. This allows the result to be stored in a variable or used in expressions.
Example:
<?php
function multiply($x, $y) {
  return $x * $y;
}
$result = multiply(4, 3);
echo "Result: $result";
?>

Output:
Result: 12

4. Function with Default Arguments:
You can assign default values to function parameters. If no value is passed, the default value will be used.
Example:
<?php
function greetUser($name = "Guest") {
  echo "Hello, $name!";
}
greetUser("Saloni");
greetUser(); // Uses default
?>

Output:
Hello, Saloni!
Hello, Guest!

5. Passing Arguments by Value and Reference:
By default, arguments are passed by value (a copy of the variable is used). To pass by reference, use an ampersand (&) before the parameter name so that changes affect the original variable.

Example (By Reference):
<?php
function increment(&$num) {
  $num++;
}
$x = 5;
increment($x);
echo $x; // Output: 6
?>

6. Variable Functions:
In PHP, you can store a function name in a variable and call it dynamically.
Example:
<?php
function sayHello() { echo "Hello!"; }
$func = "sayHello";
$func(); // Calls sayHello()
?>

7. Built-in Functions:
PHP has many built-in functions for tasks like string manipulation, mathematical calculations, and date handling.
Examples:
strlen("Hello") → Returns length of string.
sqrt(16) → Returns square root (4).
date("Y") → Returns current year.

8. User-defined vs Built-in Functions:
User-defined functions are created by the programmer, while built-in functions are provided by PHP itself. Both help simplify code and reduce redundancy.

9. Example Program:
<?php
function calculateArea($radius) {
  $area = 3.14 * $radius * $radius;
  return $area;
}
echo "Area of Circle: " . calculateArea(7);
?>

Output:
Area of Circle: 153.86

10. Summary:
Functions in PHP allow code reuse, better organization, and improved readability. They can accept parameters, return values, and even have default arguments. PHP supports both built-in and user-defined functions, making it a powerful language for modular programming.

Example Programs

Program 1: Simple User-defined Function

<?php
function greet() {
    echo "Welcome to Learn PHP!";
}
greet();
?>

Output:

Welcome to Learn PHP!

Program 2: Function with Parameters

<?php
function add($a, $b) {
    $sum = $a + $b;
    echo "Sum: $sum";
}
add(10, 20);
?>

Output:

Sum: 30

Program 3: Function with Return Value

<?php
function multiply($x, $y) {
    return $x * $y;
}
$result = multiply(5, 4);
echo "The product is: $result";
?>

Output:

The product is: 20

Program 4: Function with Default Parameter

<?php
function greetUser($name = "Guest") {
    echo "Hello, $name!";
}
greetUser("Saloni");
echo "<br>";
greetUser();
?>

Output:

Hello, Saloni!
Hello, Guest!

Program 5: Recursive Function (Factorial Example)

<?php
function factorial($n) {
    if ($n == 0) {
        return 1;
    } else {
        return $n * factorial($n - 1);
    }
}
echo "Factorial of 5 is: " . factorial(5);
?>

Output:

Factorial of 5 is: 120

Summary

In this section, you learned how PHP functions help make programs organized and reusable. Functions can take arguments, return values, and even call themselves recursively. They are extremely helpful in reducing repetitive code, improving readability, and handling specific tasks modularly.

By mastering functions, developers can create powerful, structured, and maintainable PHP applications that are easier to scale and debug.
