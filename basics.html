PHP Basics

PHP Basics introduces you to the foundation of server-side scripting. In this section, you’ll learn the basic syntax, how to print content, use variables, constants, operators, and basic data types. These concepts form the building blocks of every PHP application and help you understand how dynamic pages are generated on the web.

---
1. PHP Tokens
Definition:
Tokens are the smallest building blocks of a PHP program. PHP scripts are made up of tokens which include:
Keywords (e.g., if, else, while, for)
Variables (e.g., $a, $name)
Constants
Operators
Strings and Numbers
Punctuation Symbols (;, {}, ())
Example:
$x = 10; // Tokens: $, x, =, 10, ;
---
2. Variables
Definition:
Variables are containers used to store data temporarily in memory.
Syntax:
$variable_name = value;
Rules for Variable Names:
Must start with a $ symbol.
Must start with a letter or underscore.
Cannot start with a number.
Variable names are case-sensitive.
Example:
$name = "Saloni";
$age = 20;
---
3. Variable Scope
Definition:
Scope defines the accessibility of variables within the program.
Types of Variable Scope in PHP:
1. Local Scope:
Declared inside a function and accessible only within it.
function test() {
$x = 10;
echo $x; // Accessible
}
2. Global Scope:
Declared outside all functions and accessible globally using global keyword.
$y = 20;
function display() {
global $y;
echo $y;
}
3. Static Scope: Retains its value even after the function execution is over.
function counter() {
static $count = 0;
$count++;
echo $count;
}
---
4. Constants
Definition: Constants are identifiers for simple values that cannot be changed during script execution.
Syntax:
define("CONSTANT_NAME", value);
Example:
define("SITE_NAME", "PHP Learning");
echo SITE_NAME;
Features:
No $ sign before name.
Automatically global.
Value cannot be changed once defined. ---
5. Data Types in PHP PHP supports several data types categorized as follows:
Type Description Example
String Sequence of characters $name = "PHP";
Integer Whole numbers $num = 20;
Float/Double Decimal numbers $price = 99.99;
Boolean True or False $isValid = true;
Array Collection of values $colors = array("red","blue");
Object Instance of a class new ClassName();
NULL Variable with no value $data = NULL;
---
6. Number Handling in PHP PHP handles numbers as integers and floats automatically.
Common number functions:
is_numeric($x) → Checks if variable is number.
round($num) → Rounds to nearest integer.
rand(min, max) → Generates random number.
abs($num) → Returns absolute value.
Example:
$num = -4.7;
echo abs($num); // Output: 4.7
---
7. Operands and Operators
Operands: The variables or values on which operators perform actions.
$a = 5 + 10; // Operands: 5 and 10
Operators: Symbols that perform operations on operands.
Arithmetic Operators: +, -, *, /, %
Assignment Operators: =, +=, -=
Comparison Operators: ==, !=, >, <, >=, <=
Logical Operators: &&, ||, !
Increment/Decrement Operators: ++, --
String Operators: . (concatenation)
Array Operators: +, ==, ===, !=
Ternary Operator: condition ? value1 : value2
---
8. Comments in PHP
Purpose: Used to make code readable and explain logic.
Types of Comments:
// Single-line comment
# Another single-line comment
/* Multi-line
comment */
---
9. echo and print Statements
Feature | echo |print
Usage | Used to output one or more strings |Used to output a single string
ReturnValue| No return value |Returns 1
Speed |Faster than print |Slightly slower
Example echo "Hello PHP!"; print "Hello PHP!";
Example:
echo "Welcome to PHP!";
print "Learn PHP Basics.";
---
✨ Summary
Tokens are basic units of PHP code.
Variables store data; constants hold fixed values.
Variable scope determines accessibility.
Data types define kinds of data stored.
Operators perform actions on operands.
Comments improve code readability.
echo and print are used to display output. ---

Example Programs

Program 1: Hello World

<?php
echo "Hello, World!";
?>

Output:

Hello, World!

Program 2: Using Variables

<?php
$name = "Saloni";
$age = 20;
echo "My name is $name and I am $age years old.";
?>

Output:

My name is Saloni and I am 20 years old.

Program 3: Arithmetic Operations

<?php
$a = 10;
$b = 5;
echo "Sum: " . ($a + $b) . "<br>";
echo "Difference: " . ($a - $b) . "<br>";
echo "Product: " . ($a * $b);
?>

Output:

Sum: 15
Difference: 5
Product: 50

Program 4: Constants in PHP

<?php
define("SITE_NAME", "Learn PHP Hub");
echo "Welcome to " . SITE_NAME;
?>

Output:

Welcome to Learn PHP Hub

Program 5: Data Types Example

<?php
$integer = 25;
$float = 12.5;
$string = "PHP is fun!";
$boolean = true;

echo "Integer: $integer<br>";
echo "Float: $float<br>";
echo "String: $string<br>";
echo "Boolean: " . ($boolean ? 'true' : 'false');
?>

Output:

Integer: 25
Float: 12.5
String: PHP is fun!
Boolean: true

Summary

In this section, we learned about the basic elements of PHP — how to write simple scripts, use variables, perform arithmetic, define constants, and understand data types. These core concepts help in building logic and dynamic content for websites. Mastering PHP basics is the first step toward learning advanced concepts such as functions, arrays, and database integration.
