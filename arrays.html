Arrays in PHP

Arrays in PHP are special variables that can hold multiple values under a single name. They are extremely useful when you need to store and access a collection of related data efficiently.

PHP supports three main types of arrays: Indexed arrays (numeric keys), Associative arrays (named keys), and Multidimensional arrays (arrays within arrays). Each serves different data handling needs.

Arrays are one of PHP’s most powerful features, allowing developers to store lists, manage data structures, and perform complex operations using built-in functions like count(), array_merge(), and sort().

An array in PHP is a special variable that can hold multiple values under a single name. Each value in an array is called an element, and each element has a key (index) used to access it.

Arrays are used to store multiple related values in one variable instead of many individual variables. For example, instead of creating multiple variables for each fruit name, we can store them all in a single array.

PHP supports three main types of arrays: Indexed Arrays, Associative Arrays, and Multidimensional Arrays.

1. Indexed Array: These use numeric indexes starting from 0. Example:
$fruits = array("Apple", "Banana", "Cherry");
echo $fruits[1]; // Output: Banana

2. Associative Array: These use named keys instead of numbers. Example:
$age = array("John" => 25, "Riya" => 20, "Aman" => 22);
echo $age["Riya"]; // Output: 20

3. Multidimensional Array: These contain one or more arrays inside another array and are used to store data in tabular form. Example:
$students = array(
  array("Riya", 18, "A+"),
  array("Aman", 19, "A"),
  array("John", 17, "B+")
);
echo $students[0][0]; // Output: Riya

PHP provides several built-in array functions. Common ones include:
count($arr) – Returns the number of elements.
array_push($arr, $val) – Adds an element at the end.
array_pop($arr) – Removes the last element.
array_merge($a1, $a2) – Merges two or more arrays.
array_keys($arr) – Returns all the keys of an array.
sort($arr) – Sorts an array in ascending order.

Traversing Arrays: You can display all elements of an array using loops.
Example for indexed array:
foreach ($fruits as $fruit) { echo $fruit . "<br>"; }
Example for associative array:
foreach ($age as $name => $years) { echo "$name is $years years old.<br>"; }

Example Program:
$marks = array("Math" => 90, "Science" => 85, "English" => 88);
foreach ($marks as $subject => $score) { echo "$subject : $score <br>"; }
Output:
Math : 90
Science : 85
English : 88

Advantages of Arrays: Arrays store multiple values in a single variable, are easy to manage, reduce code length, and are flexible and dynamic.

Summary: Arrays in PHP store multiple values in one variable. They are of three types – Indexed, Associative, and Multidimensional. Array functions make it easy to add, remove, or sort elements. Loops like foreach() are used to access and display array elements efficiently.

Example Programs

Program 1: Indexed Array Example

<?php
$fruits = array("Apple", "Banana", "Mango");
echo "I like " . $fruits[0] . ", " . $fruits[1] . " and " . $fruits[2] . ".";
?>

Output:

I like Apple, Banana and Mango.

Program 2: Associative Array Example

<?php
$marks = array("Saloni" => 90, "Riya" => 85, "Karan" => 88);
echo "Saloni scored " . $marks["Saloni"] . " marks.";
?>

Output:

Saloni scored 90 marks.

Program 3: Multidimensional Array Example

<?php
$students = array(
    array("Riya", 18, "A"),
    array("Saloni", 19, "B"),
    array("Karan", 18, "A")
);

echo $students[1][0] . " is " . $students[1][1] . " years old and got grade " . $students[1][2];
?>

Output:

Saloni is 19 years old and got grade B

Program 4: Traversing Array using foreach

<?php
$colors = array("Red", "Green", "Blue");
foreach ($colors as $value) {
    echo "$value <br>";
}
?>

Output:

Red
Green
Blue

Program 5: Sorting an Array

<?php
$numbers = array(4, 2, 8, 6);
sort($numbers);
foreach ($numbers as $num) {
    echo "$num ";
}
?>

Output:

2 4 6 8

Summary

Arrays are vital for storing and managing multiple data values efficiently in PHP. With indexed, associative, and multidimensional arrays, you can represent real-world data easily.

Understanding how to loop through, sort, and modify arrays is crucial for any PHP developer. Mastering array handling improves both the performance and structure of your web applications.
